Setting Up Everything

Installing Unity and Visual Studio

  • Go to https://unity.com/download#how-get-started and download the unity hub installer.
  • Sign in or create an account if you don't have one and install the editor.
  • A download of the community edition of Visual Studio will be included, but you could also Visual Studio Code.
  • If you're using Visual Studio Code, go to extensions "Ctrl+Shift+X" and download the extension C# made by Microsoft.

Creating Your First Project and Looking at the UI

  • Go to Projects and select New Project. Create a new 2D Project with the Built-In Render Pipeline, name it whatever you want.
  • Here's some important features of Unity's UI.
    1. Scene View: This is where you design and arrange your game's elements in a 3D or 2D environment.
    2. Game View: Allows you to see how your game will look and behave to players.
    3. Hierarchy: Displays a list of all the objects in your current scene.
    4. Inspect: Provides ddetailed information about the currently selected object allowing you to edit its properties.
    5. Project Window: Displays all the assets in your project, such as scripts, textures, and models.
    6. Asset Store: Access to a vast library of free and paid assets, including scripts, textures, and models to enhance your game.
    7. Toolbar: Contains various tools for manipulating objects, navigating the scene, and controlling the editor's view.
