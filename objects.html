Objects and Components

How to Create Objects?

  • Right click on the Hierarchy to open the menu.
  • Select "Create Empty" to create a new empty object, or choose from the predefined objects like "2D Object > Sprites > Square".
  • You can organize your Hierarchy, by using an empty game object to put in all related object into one group (e.g., creating an "Enemies" game object to group all enemies).

How to Add Components?

  • Select the object and go to the Inspector.
  • Go to the very bottom of the Inspector and click on "Add Component".
  • Search for the component that you wanna add. Here's some of the components that you'll be using.
    1. Transform (automatically added by default): Allows you to change the position of the object.
    2. Sprite Renderer: Used for rendering an image for 2D objects.
    3. Rigidbody2D: Gives the object physics.
    4. Collider2D: Enables collision detection.
    5. Animator: Controls animations for the object.
    6. Audio Source: Plays sounds attached to the object.
    7. Camera: Defines the player's view in the game.
    8. Light: Adds lighting effects to the scene.
    9. Script: Adds custom behavior through C# scripts.
