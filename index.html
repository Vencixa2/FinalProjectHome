Getting Started With C# and Unity

What is C#?

  • C# (pronounced C-sharp) is a programming language that uses syntax similar to other popular languages like C, C++, and Java.
  • It is Primarily used for creating Windows applications, web services, and enterprise software.
  • In the context of Unity, C# is the primary scripting language used to create game logic, handle user input, and control game objects and behaviours.
  • Its features include strong typing, automatic garbage collection, and a rich library of built-in functions, making it an excellent choice for game development.

What is Unity?

  • Unity is a widely-used game development engine/platform that allows developers to create 2D, 3D, VR, and AR games and experiences. It provides loads of tools
  • It provides a comprehensive set of tools for designing, developing, and deploying games across various platforms, including consoles, PCs, mobile devices, and web browsers.
  • Unity's natural interface, extensive asset store, and strong community support make it a popular choice for both begginer and experienced game developers. By using C# in Unity, developers can write scripts that control game mechanics. interact with game objects, and implement complex gameplay features.
